# Cosmasys Website

Enterprise-grade website for Cosmasys - Large-scale systems for human skill intelligence and robotic training.

## Overview

Cosmasys builds large-scale systems for capturing, structuring, and delivering human skill intelligence for robotics and embodied AI. This website presents our platform, services, and capabilities in a professional, enterprise-ready format.

## Features

- **Modern, Responsive Design**: Fully responsive layout that works on all devices
- **Enterprise-Grade UI**: Clean, professional design with smooth animations
- **Smooth Navigation**: Sticky navigation with smooth scrolling
- **Contact Form**: Interactive contact form for inquiries
- **Performance Optimized**: Lightweight and fast-loading

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **JavaScript**: Vanilla JS for interactivity
- **Google Fonts**: Inter font family for professional typography

## File Structure


## Getting Started

1. Clone the repository:
```bash
git clone <repository-url>
cd website
```

2. Open `index.html` in a web browser, or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

3. Navigate to `http://localhost:8000` in your browser

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1a1a2e;
    --secondary-color: #16213e;
    --accent-color: #0f3460;
    /* ... */
}
```

### Content
All content is in `index.html`. Simply edit the HTML to update text, sections, or structure.
